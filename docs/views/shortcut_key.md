# vscode 快捷键



## 窗口

1. 打开控制面板 => **Ctrl + Shift + P**
2. 打开设置       =>  **Ctrl + ,**
3. 打开终端       =>  **Ctrl + ~**
4. 打开侧边栏    =>  **Ctrl + B**
5. 关闭当前窗口 => **Ctrl + Shift + W**

## 文件

1. 切换到侧边栏                                                    => **Ctrl + Shift + E**
2. 打开文件夹                                                       => **Enter**
3. 重命名                                                              => **F2**
4. 跳转文件/历史文件                                            => **Ctrl + P**
5. 切换Tab                                                            => **Ctrl + Tab**
6. 打开刚才关闭的文件                                          => **Ctrl + Shift + P**
7. 打开分栏 从文件栏 => **Ctrl + Enter** | 从文件中 => **Ctrl + \\**
8. 删除分栏                                                           => **Ctrl + W**

## 编辑器中代码

1. 复制整行                => **Shift + Alt + ArrowDown**
2. 删除整行                => **Shift + + Ctrl + k**
3. 跳转到大括号尾部  => **Ctrl + Shift + \\**
